# Prompt 10 - Resume-Mentor V3

Version 1.0

## Role Council

You are a redesign council made of:

- PM
- CEO
- Investor
- Recruiter
- Designer
- Engineer
- Growth Hacker
- Career Coach

Your task is to redesign Resume-Mentor as the world's strongest Career Operating System.

Do not design it as:

- a resume tool
- a resume template website
- an ATS checker

## Product Vision

Resume-Mentor V3 is a Career Operating System that helps users capture, structure, improve, and activate their career capital every day.

It is not a document generator.

It is a long-term career memory, strategy, and execution system.

Positioning:

```text
Build your Career Operating System.
Turn every project, win, lesson, skill, and career move into reusable career capital.
```

Core belief:

```text
A resume is only one export from a much larger career knowledge system.
```

Resume-Mentor V3 should feel like a mix of:

- Notion for career memory

## Core Features

### 1. Career Knowledge Base

The core database that stores:

- Career Profile
- Achievement Database
- Leadership Database
- Skill Evidence Map
- Interview Story Bank
- Promotion Evidence
- Learning Roadmap
- Salary Proof
- Portfolio Proof

Why it matters:

Users stop rewriting from scratch. Every future output comes from the same source of truth.

### 2. Daily Career Capture

A lightweight daily habit:

```text
What did you ship today?
What problem did you solve?
Who did you help?
What metric moved?
What did you learn?
What proof can you save?
```

This turns ordinary work into future resume, promotion, and interview material.

### 3. Career Score

A diagnostic score, not a hiring probability.

Dimensions:

- Career Clarity
- Evidence Strength
- Skill Proof
- Leadership Signal
- Market Fit
- Interview Readiness
- LinkedIn Visibility
- Compensation Leverage

### 4. Goal-Based Career Operating Modes

Users choose a mode:

- Get a job
- Switch career
- Get promoted
- Prepare for interviews
- Build executive brand
- Move overseas
- Negotiate salary
- Become founder/operator

Each mode activates different agents, skills, prompts, dashboards, and outputs.

### 5. Career Asset Generator

Generates exports from the Career Knowledge Base:

- Resume
- LinkedIn Profile
- Interview Pack
- Cover Letter
- Portfolio Case Study
- GitHub Profile Story
- Promotion Packet
- Salary Negotiation Script
- Executive Bio
- Headhunter Pitch

### 6. Interview Gym

Daily and role-specific practice:

- STAR drills
- Leadership drills
- Product sense drills
- Executive presence drills
- Recruiter screen drills
- Bar raiser drills
- Company-specific simulations

### 7. Career Gap Engine

Compares current state to target role:

- Skill gaps
- Evidence gaps
- Leadership gaps
- Compensation gaps
- Brand gaps
- Network gaps

Outputs:

- 90-day plan
- 180-day plan
- 365-day plan

### 8. Opportunity Tracker

Not a generic job tracker.

Tracks:

- Target companies

- Relationship map

- Follow-up tasks
- Feedback loops
- Learning gaps from rejection

### 9. Proof Vault

Stores proof assets:

- GitHub repos
- Case studies
- Presentations
- Performance reviews
- Testimonials
- Public posts

### 10. Agent and Skill Marketplace

Users assemble their own Career Mentor Team and workflows.

## Moat

### 1. Career Knowledge Graph

The more a user records work, wins, feedback, skills, interviews, and goals, the harder it is to leave.

### 2. Evidence-Based Career Memory

Resume-Mentor owns structured proof, not generic text.

### 3. Skill Marketplace

Community and expert-built skills create long-tail value across industries.

### 4. Agent Marketplace

Users personalize their Career Mentor Team, creating workflow lock-in.

### 5. Feedback Loop

Applications, interviews, rejections, promotions, salary outcomes, and feedback improve the user's Career OS.

### 6. Privacy-First Trust

If Resume-Mentor becomes trusted for private career data, switching cost increases.

### 7. Daily Habit

Daily career capture turns Resume-Mentor into a system of record.

## Why Users Return Every Day

Users do not return daily to edit resumes.

They return daily because Resume-Mentor becomes their career cockpit.

Daily use cases:

```text
1. Log today's work win.
2. Capture a metric before it is forgotten.
3. Save proof from a project.
4. Turn a meeting outcome into a leadership story.
5. Practice one interview question.
6. Update progress toward target role.
7. Review today's career gap action.
8. Prepare manager update or promotion evidence.
9. Improve LinkedIn or public proof.
10. Track application or interview feedback.
```

Daily loop:

```text
Capture
→ Challenge
→ Store
→ Improve Score
→ Generate Asset
→ Take Next Action
```

Retention mechanics:

- daily career capture streak
- weekly career score update
- interview practice streak
- target-role progress meter
- proof vault completion
- promotion evidence tracker
- agent-generated daily challenge
- weekly career review
