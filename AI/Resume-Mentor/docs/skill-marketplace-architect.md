# Resume-Mentor Skill Marketplace Architect

Version 1.0

## Marketplace Goal

Resume-Mentor Skill Marketplace lets users assemble reusable career skills that transform a Career Knowledge Base into role-specific outputs: resumes, LinkedIn profiles, interview answers, leadership stories, promotion packages, portfolios, salary negotiation scripts, and long-term career plans.

## Skill Schema

Each skill follows this contract:

```text
Name:
Goal:
Input:
Output:
Challenge Rule:
Review Rule:
Success Criteria:
```
