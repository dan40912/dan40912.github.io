## 🧠 學習進度（核心概念）
> 下方是 Obsidian **Bases** 視圖（已啟用核心外掛）。依成熟度 status 一眼看出哪些讀過了。

```base
filters:
  and:
    - 'type == "concept"'
views:
  - type: table
    name: 核心概念
    order:
      - file.name
      - status
      - level
      - updated
```

## 📰 最近快訊
```base
filters:
  and:
    - 'type == "news"'
views:
  - type: table
    name: 每日快訊
    order:
      - file.name
      - updated
    sort:
      - property: updated
        direction: DESC
    limit: 14
```

## 🗓️ 每週精選
```base
filters:
  and:
    - 'type == "weekly"'
views:
  - type: table
    name: 每週精選
    order:
      - file.name
      - updated
    sort:
      - property: updated
        direction: DESC
```

---
> 💡 若上方 `base` 區塊顯示錯誤，多半是你的 Obsidian 版本 Bases 語法略有差異——可點區塊右上角用 GUI 編輯，或先忽略，手動導覽連結一樣可用。
