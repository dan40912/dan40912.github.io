# OpenClaw 個人 AI 助理 / OpenClaw（龍蝦 🦞）

> **一句話：** OpenClaw 是開源的「**個人 AI 助理**」——跑在你自己的裝置上，接你已經在用的通訊軟體（WhatsApp、Telegram、Slack、Discord、iMessage…），而且能真的在你機器上「動手做事」。標語「**The lobster way 🦞**」就是「龍蝦」的由來。

> 由 PSPDFKit 創辦人 Peter Steinberger 開發，暱稱 **Molty**（前身 clawdbot／moltbot），釋出後 72 小時破 6 萬 GitHub stars。資料以 2026-06 來源為準。

## 1. 核心特點 Features
- **本機執行、隱私優先**：跑在你自己的裝置，自帶 API key；可用雲端模型或全本機模型（[[Model Agnostic 模型無關]]）。
- **跨通訊軟體**：WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、Teams、LINE、WeChat… 在你慣用的對話框裡指揮它。
- **跨對話記憶**：記得脈絡（[[Context 脈絡與記憶]]）。
- **100+ AgentSkills**：可執行 shell 指令、管理檔案系統、做網頁自動化——對應 [[Skill 技能]] 與 [[Tool Use 工具呼叫]]。
- 本質是一個會「規劃→行動」的 [[Agent 代理]]，被稱為「最接近 JARVIS」的開源助理。

## 2. 怎麼用 / 對你的意義
- 想要一個**自己的、跑在本機、接你日常聊天 App** 的助理，OpenClaw 很適合。
- 因為它能執行 shell 與操作檔案，**[[Guardrails 護欄]] 特別重要**：限制可用工具與權限、對高風險操作設確認，避免越權。

## 3. 與 Hermes Agent 的差異
- **[[Hermes Agent（Nous Research）]]**：強調「自我進化學習迴圈」，偏研究／可成長 Agent。
- **OpenClaw**：強調「個人、跨通訊軟體、跑本機做事」，偏日常personal assistant。
