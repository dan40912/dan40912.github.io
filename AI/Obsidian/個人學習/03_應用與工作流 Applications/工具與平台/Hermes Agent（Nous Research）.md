# Hermes Agent / Nous Research Hermes

> **一句話：** Hermes 是 Nous Research 的開源產物，分兩個層次——**Hermes 模型系列**（開源 LLM）與更受矚目的 **Hermes Agent**：一個**會自我進化**的開源自主 AI Agent，有內建學習迴圈，用越多越聰明。

> 資料以 2026-06 來源為準；版本演進快，請再查官方。

## 1. 兩個 Hermes 別搞混
- **Hermes 模型**：開源 LLM 系列。Hermes 4.3（2025-08）建於 ByteDance 的 Seed 36B 基座，以聯動使用、少審查著稱。
- **Hermes Agent**：2026-02-25 釋出的開源自主 Agent 框架，4 個月內衝破 18 萬 GitHub stars，是 2026 成長最快的開源 Agent 框架。

## 2. Hermes Agent 核心特點
- **內建學習迴圈（self-improving）**：不只完成任務，還會**記住、累積可重用的 skills、越用越強**——對應 [[Skill 技能]]、[[Self-Reflection 自我反思]]、[[Feedback Loop 回饋迴圈]] 與長期 [[Context 脈絡與記憶]]。
- **Model-agnostic**：不綁單一模型（[[Model Agnostic 模型無關]]）。
- **多介面**：Windows／Mac／Linux 桌面 App、Web UI、CLI、終端機介面；2026-06-02 推出官方桌面 App（v0.15.2，串流工具輸出）。
- **可自架、隱私可控**；v0.7.0（2026-04）加入可插拔記憶供應者、憑證輪替、反偵測瀏覽器等。

## 3. 怎麼用 / 對你的意義
- 想要一個**自架、會自我學習**的個人或研究 Agent，Hermes Agent 是當前最熱選項之一。
- 它把本庫「[[Agent 代理]] + [[Skill 技能]] + [[Self-Reflection 自我反思]] + 記憶」實作成一個會長期成長的系統，是觀摩 agentic 設計的好範本。
